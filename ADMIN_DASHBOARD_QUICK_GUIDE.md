## ğŸ“Š Understanding Charts

### Line Chart (Collection Trends)

```
Shows item collection volume over 7 days
Y-axis: Number of items
X-axis: Days (Mon-Sun)
Green line: Upward trend = good collection rate
```

### Pie Chart (Status Distribution)

```
Shows breakdown of item statuses
Orange slice: Pending items
Blue slice: Assigned items
Green slice: Delivered items
Percentages shown for each
```

### Bar Chart (Category Distribution)

```
Shows items by category
X-axis: Item categories
Y-axis: Number of items
Green bars: Higher bars = popular categories
```

## ğŸ†˜ Troubleshooting

### Search Not Finding Users

- Check spelling of name
- Ensure full name matches exactly
- Clear search and try again

### Data Not Updating

- Refresh page (F5)
- Check internet connection
- Log out and log back in
- Contact support if persists

### Chart Not Loading

- Enable JavaScript
- Try different browser
- Clear browser cache
- Restart application

### Mobile Tables Not Scrolling

- Use two-finger swipe to scroll horizontally
- Rotate device to landscape
- Use desktop view for better table viewing
